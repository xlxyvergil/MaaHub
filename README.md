# MaaHub

中文 | [English](./README.en.md)

MaaHub 是一个面向社区协作的仓库，目标是沉淀 Maa 相关的 `Skills`、`Pipelines`、`Customs` 与 `Experiences`，并在后续通过网站把这些内容整理成可浏览、可检索、可贡献的资源中心。

当前仓库仍处于构建阶段，敬请期待！

## 仓库结构

```text
.
|-- Storage/
|   |-- customs/
|   |-- experiences/
|   |-- pipelines/
|   `-- skills/
|-- website/
|-- README.md
`-- README.en.md
```

### 结构说明

| 路径                   | 作用                         |
| ---------------------- | ---------------------------- |
| `Storage/customs/`     | 用于收纳社区自定义代码条目   |
| `Storage/experiences/` | 用于收纳教程、经验与案例说明 |
| `Storage/pipelines/`   | 用于收纳 Pipeline 相关条目   |
| `Storage/skills/`      | 用于收纳 Skill 条目          |
| `website/`             | 站点所在的 Astro 工作区      |

## 贡献方式

不建议现在进行贡献，仓库正在架构阶段，敬请期待！
