# PR Template Skill

This directory is the official MaaHub pull request template for skill contributions.

## Purpose

- Show the expected `maahub_meta.json` layout for website metadata.
- Show the expected `SKILL.md` layout for skill frontmatter and instructions.
- Provide optional `references/` and `scripts/` examples.

## Files

- `maahub_meta.json`: MaaHub website metadata
- `README.md`: human-readable introduction for contributors
- `SKILL.md`: canonical skill definition
- `references/checklist.md`: optional reference material
- `scripts/example.sh`: optional helper script
